GeneNoteBook

A collaborative notebook for genes and genomes

Getting started
  • Download GeneNoteBook v0.1-beta

  • Install dependencies

    apt-get install nodejs mongodb
    Linux
    brew install nodejs mongodb
    OSX

  • Start mongodb daemon

    mongod

  • Start GeneNoteBook

    genenotebook.sh settings.json

  • Load data

    node scripts/add_reference.js
