Running a server

GeneNoteBook needs a running MongoDB daemon.
To start a MongoDB daemon with default settings on port 27017, using <data folder> as storage location:

> mongod --fork --dbpath <data folder> --logpath mongod.log
Note: <data folder> must be an existing folder

Local server Docker Apache configuration
To run a GeneNoteBook server with default settings:
> genenotebook run
Alternatively, to run GeneNoteBook in the background:
> genenotebook run 2>&1 genenotebook.log &
This starts a GeneNoteBook server on the default port (3000) and adds a default admin user account. Check http://localhost:3000 and log in with the default admin account (username: admin, password: admin).
Immediately change the default password and/or username of the admin account!
http://localhost:3000/profile
Running GeneNoteBook from the Docker container is similar to running a default installation:
> docker container run quay.io/biocontainers/genenotebook:0.1.8--h2d50403_0 genenotebook run
