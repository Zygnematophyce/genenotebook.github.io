Installation

Genenotebook requires Node.js, MongoDB and NCBI BLAST to run.
By using either bioconda or docker these dependencies can be installed automatically.
Alternatively, install these dependencies yourself and grab the GeneNoteBook tarball from GitHub.
Bioconda Github Docker
By using the conda package management system GeneNoteBook and its dependencies can be installed as follows:
> conda install -c bioconda genenotebook
Alternatively, to isolate your installation from other packages, use a conda environment:
> conda create -c bioconda -n gnb_env genenotebook
If you want to bundle GeneNoteBook from source code, or run a development server, you need Node.js (>= 8.0.0), MongoDB (>= 3.4.0) and NCBI BLAST (>= 2.5.0).
Clone git repository
> git clone https://github.com/genenotebook/genenotebook
Bundle the source code
> npm run complete_build
To run a development server, first install javascript dependencies with npm install and then start a server with npm run dev

> docker pull genenotebook/genenotebook:v0.1.13
GeneNoteBook Docker files are hosted in a separate repository: https://github.com/genenotebook/genenotebook_docker
