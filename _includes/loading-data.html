Loading data

Once a GeneNoteBook server is running data can be added through a command line interface.
For testing purposes a test dataset is available.

Genome / Annotation InterProScan Orthogroups Gene expression
First add the reference genome:
> genenotebook add genome -u <username> -p <password>
-n <genome_name> <genome.fasta>
Then add the corresponding genome annotation:
> genenotebook add annotation -u <username> -p <password>
-n <genome_name> <annotation.gff3>
To enable the BLAST search for the added annotation, browse to the admin settings and navigate to 'Genomes' --> 'Edit' --> 'Make Blast DB'.
Adding InterProScan results:
> genenotebook add interproscan -u <username> -p <password>
<InterProScan gff3 output file>
Adding OrthoFinder results:
> genenotebook add orthogroups -u <username> -p <password>
<OrthoFinder results folder>
Adding gene expression quantification (kallisto results):
genenotebook add transcriptome -u <username> -p <password>
<Kallisto abundance.tsv file>
